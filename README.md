# Vansh Sharma Portfolio

A premium recruiter-focused Full Stack Developer portfolio built with React, Vite, Tailwind CSS, Framer Motion, React Router, React Icons, EmailJS, and the GitHub API.

## Quick Start

```bash
npm install
npm run dev
```

## Configure Personal Details

- Update profile content in `src/data/profile.js`.
- Update project cards in `src/data/projects.js`.
- Place your resume at `public/resume.pdf` or set `VITE_RESUME_URL`.
- Add EmailJS keys in `.env` using `.env.example` as the template.

## Build

```bash
npm run build
npm run preview
```
